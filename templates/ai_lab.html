学习路径

4阶段 · 8周突击计划

按桑鸿乾老师制定的学习路线，每周投入充足时间，配合AI工具高效学习

第1-2周
🐍

Python编程强化

  • 在"菜鸟教程"上复习Python基础语法
  • 理解变量、函数、类、模块等核心概念
  • 掌握NumPy/Pandas数据处理基础
  • 能阅读并理解AI生成的Python代码
  • 使用Claude Code辅助编写和调试代码
推荐工具： 菜鸟教程 Claude Code
第3-4周
🌐

Web前端开发

  • HTML：常用标签、语义化、表单
  • CSS：选择器、盒模型、Flexbox/Grid布局
  • JavaScript：DOM操作、事件处理、fetch API
  • 制作一个个人主页作为练习项目
  • 理解"客户端本质是套壳浏览器"
推荐工具： HTML教程 CSS教程 JS教程
第5-6周
☁️

服务器部署

  • 在阿里云注册账号，购买学生优惠服务器
  • 学习Linux基本命令和文件操作
  • 使用SSH远程连接云服务器
  • 安装配置Nginx + Python环境
  • 使用Gunicorn部署Flask应用
  • 了解Cloudflare Pages/Workers部署方式
推荐工具： 阿里云ECS Cloudflare
第7-8周
🤖

AI基础认知 + 工具链

  • 理解大语言模型（LLM）基本原理
  • 掌握Transformer架构和"幻觉"成因
  • 熟练使用ChatGPT、Claude Code、Gemini
  • 了解RAG、Agent、Function Calling概念
  • 维护GitHub作为学习日志和简历
  • 学会向AI提供充分背景信息获取优质回答
推荐工具： ChatGPT Gemini GitHub
